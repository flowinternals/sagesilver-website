# SageSilver Website

Public website for SageSilver Pty Ltd, built with Next.js, React, TypeScript and Tailwind CSS.

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production

```bash
npm run build
npm start
```

The project can be deployed directly to Vercel or any Node.js host supporting Next.js 16 and Node.js 20.9 or newer.

## Current limitations

- The contact form is visual only and does not submit data.
- Placeholder telephone numbers and demonstration testimonials must be replaced before launch.
- Images and video are currently loaded from third-party CDN URLs.
